## 🔄 إصلاحات المزامنة بين الأجهزة

### ✅ حل مشكلة اختلاف البيانات بين الأجهزة
- **قبل:** `saveArray()` كانت فقط تضيف/تحدّث ولا تحذف — الطلاب المحذوفون يبقون في قاعدة البيانات
- **بعد:** `replaceCollection()` تحذف السجلات المحذوفة أيضاً

### ✅ حل مشكلة السنوات الدراسية المختلفة
- **قبل:** كل جهاز يبدأ بالسنوات الافتراضية المكتوبة في الكود
- **بعد:** السنوات تُحفظ في Supabase عند أول استخدام — كل الأجهزة تقرأ نفس البيانات

### ✅ إزالة كل الحفظ المحلي
- تسجيل الدخول: لم يعد يقرأ من localStorage
- ترتيب البلاطات: يُحفظ في Supabase
- إيصالات الطباعة: تستقبل الإعدادات كـ prop
- لوحة التشخيص: أُصلحت لاستخدام Supabase

## 📅 تقدم السنة الدراسية تلقائياً (جديد)

### كيف يعمل:
- **في 1 أغسطس من كل سنة** يتحول النظام تلقائياً للسنة الدراسية الجديدة
- مثال: في 1 أغسطس 2026 → يتحول من "2025-2026" إلى "2026-2027"
- إذا السنة الجديدة غير موجودة في القائمة — يُنشئها تلقائياً
- التقدم يحدث عند فتح التطبيق (ليس بشكل مستمر في الخلفية)
- يُحفظ التغيير في Supabase فوراً — كل الأجهزة تتحدث

### القاعدة:
```
الشهر >= 8 (أغسطس) → السنة الدراسية = السنة الحالية - السنة القادمة
الشهر < 8           → السنة الدراسية = السنة الماضية - السنة الحالية
```

## 🆕 الميزات الجديدة

### 1. سحب وصل الطالب
- زر "سحب" برتقالي بجانب كل وصل
- نافذة بسبب السحب (ترك المدرسة، نقل، خطأ، الخ)
- الوصل المسحوب يظهر مشطوب مع شارة "مسحوب"
- المبلغ المسحوب لا يُحسب ضمن المدفوعات

### 2. سجل الموظف المالي الكامل
- عرض كل الرواتب والسلف لكل موظف
- ملخص مالي (إجمالي الرواتب + السلف + الكلي)
- عرض فاتورة كل عملية

### 3. سلفة الموظف
- زر "سلفة" بجانب كل موظف
- نافذة لإدخال المبلغ والتاريخ والملاحظات
- السلفة تظهر تحت الراتب في الجدول
- تُسجل كمصروف بنوع "سلفة"

## 🔍 إصلاحات التناسق

### ✅ توحيد تنسيق الأرقام
- **قبل:** 3 تنسيقات مختلفة (en-US, ar-IQ, en-IQ)
- **بعد:** `formatIQD()` واحد في كل مكان — أرقام إنجليزية موحدة

### ✅ توحيد رقم الوصل
- **قبل:** بعض الشاشات تعرض "0001" وأخرى تعرض "3456" لنفس الوصل!
- **بعد:** `formatReceiptNo()` واحد — دائماً `padStart(4, "0")`

### ✅ إصلاح التواريخ الثابتة
- **قبل:** `const currentYearNum = 2026` و `const currentMonthNum = 5` (مكتوبة يدوياً!)
- **بعد:** `new Date().getFullYear()` و `new Date().getMonth() + 1` (ديناميكية)
